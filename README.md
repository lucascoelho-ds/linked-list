# linked-list em Python

## Descrição
O projeto consiste em implementar uma lista encadeada utilizando a linguagem Python. A atividade consiste em implementar uma função count_nodes, que recebe uma lista encadeada como parâmetro e retorna o número de nós presentes na lista. A função percorre a lista encadeada usando um loop enquanto incrementa um contador. Ao final do percurso, o valor do contador é retornado.

Alguns elementos são adicionados usando o método append. Em seguida, a lista é impressa e a função count_nodes é chamada passando a lista encadeada como argumento. O resultado é então impresso na tela, exibindo o número de nós presentes na lista.

## Tecnologias Utilizadas
- Python
- VS Code

## O que o programa faz?
Este projeto implementa uma Lista Encadeada Simples em Python

O programa permite:

1. Criar nós contendo dados.
2. Conectar os nós por meio de referências ('next').
3. Adicionar novos nós ao final da estrutura.
4. Percorrer toda a lista para exibir seus elementos.
5. Contar quantos nós estão armazenados na lista.

Exemplo de saída:

10 -> 20 -> 30 -> 50 -> None
Número de nós: 4
